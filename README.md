## 🧠 Mental Health in Tech Survey – Data Analysis

### 📌 Overview
This project focuses on analyzing the OSMI Mental Health in Tech Survey dataset to understand factors affecting mental health in the workplace.

The project covers data cleaning, preprocessing, and exploratory data analysis (EDA) to extract meaningful insights.

### 🎯 Objectives
Identify and handle missing values
Remove duplicate records
Standardize and clean dataset
Detect and remove outliers
Perform exploratory data analysis
Generate insights on mental health trends

### 🛠️ Tech Stack
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook

### 📂 Dataset
OSMI Mental Health in Tech Survey Dataset
Format: CSV

### ⚙️ Project Workflow

#### 1️⃣ Data Loading
Imported dataset using Pandas
Displayed structure and preview

#### 2️⃣ Data Cleaning
Removed irrelevant columns
Handled missing values (median, mode, "Unknown")
Removed duplicate rows
Standardized column names
Fixed inconsistent values (e.g., gender)
Converted data types

#### 3️⃣ Outlier Detection
Used Boxplot for visualization
Applied IQR method for removal

#### 4️⃣ Exploratory Data Analysis (EDA)
* Age distribution
* Gender distribution
* Treatment analysis
* Family history impact
* Work interference
* Remote work effect
* Correlation heatmap

### 📊 Key Insights
🧠 Mental health awareness is increasing
🧬 Family history strongly influences treatment
👨 Dataset shows gender imbalance
💼 Mental health affects workplace productivity
🏠 Remote work has mixed impact
